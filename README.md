# 🏡 Airbnb Clone - Full Stack Project

A full-stack Airbnb-inspired web application that allows users to explore listings, make bookings, and manage their properties. Built with the **MERN Stack** (MongoDB, Express.js, React.js, Node.js) with a strong focus on **backend development and RESTful APIs**.

## 🔗 Live Demo
[Deploy Link (if available)](https://airbnb-1-ntbp.onrender.com/)

---

## 🚀 Features

- 🧾 User authentication (register/login)
- 📌 Browse listings with filtering and sorting
- 🏠 Host property listings with details and images
- 📅 Book and manage reservations
- 🔐 Secure APIs and route protection
- 📦 MongoDB for scalable data storage
- - 📲 Responsive UI using React and modern styling
- Centralized error handling with custom middleware
- Schema validation for input data

---

## 🛠️ Tech Stack

**Frontend:**
- EJS
- Axios for API requests
  
**Backend:**
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- RESTful API Design
